# 金蝶产品文档写作助手

> 帮助撰写和优化金蝶产品文档的技能

## 简介

金蝶产品文档写作助手是一款专为金蝶各产品线设计的智能写作工具，帮助您高效创作符合规范的产品文档。无论是从零开始撰写新文档，还是润色优化已有内容，都能提供专业支持。

## 功能特性

### 核心能力

| 能力 | 说明 |
|------|------|
| 从零创作 | 根据需求自动生成符合金蝶规范的文档框架和内容 |
| 润色优化 | 改进已有文档的结构、语言、格式，提升可读性 |
| 规范检查 | 自动检查文档是否符合金蝶写作规范 |
| 模板引导 | 提供14种文档类型的专业模板和写作指导 |

### 支持的文档类型

- **整体介绍** - 帮助用户全面了解模块的核心业务场景、核心功能及价值
- **功能介绍** - 帮助用户了解功能的作用、应用场景、关键字段解释
- **场景指引** - 帮助用户了解特定场景的操作步骤和注意事项
- **常见问题** - 问题分析类或知识解答类文档
- **听产品经理说** - 新特性文章，面向用户宣传新特性价值
- **实践案例** - 基于典型客户形成的案例
- **重要通知** - 产品重要公告内容
- **新特性公告** - 详细介绍最新功能特性
- **快速入门** - 帮助用户快速上手使用
- **安装部署指南** - 软件或系统的安装与配置流程
- **开发指南/API/SDK** - 为定制化开发者提供开发指导
- 以及更多...

### 触发方式

### 交互流程

```
确认文档类型 → 确认写作场景 → 提供参考材料 → 确认写作细节 → 需求确认 → 生成内容 → 确认调整 → 选择输出格式
```

1. **确认文档类型** - 选择要撰写的文档类型
2. **确认写作场景** - 从零创作或润色优化
3. **提供参考材料** - 上传 PRD、设计稿、截图等（可选）
4. **确认写作细节** - 明确产品模块、目标受众、特殊要求
5. **需求确认** - 查看任务摘要并确认
6. **生成内容** - AI 根据金蝶规范生成或优化文档
7. **确认调整** - 查看内容，支持多轮迭代修改
8. **选择输出格式** - 导出为 Markdown、Word 或纯文本

## 写作规范来源

本技能基于以下金蝶官方写作规范构建：

- **金蝶产品手册中文写作规范 V3.0**
- **金蝶产品手册中文写作规范 V2.1**

涵盖标题规范、句子规范、段落规范、列表规范、表格规范、图形规范、标点符号规范等完整写作标准。
